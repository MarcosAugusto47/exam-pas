PAS Vision

Capacitando Estudantes com Machine Learning Insights

O PAS Vision é uma ferramenta que utiliza o poder do XGBoost, um avançado modelo de aprendizado de máquina, para prever a probabilidade de aprovação no Programa de Avaliação Seriada (PAS), para ingresso na Universidade de Brasília, dado o conjunto de dados das notas nas três etapas do vestibular, se é elegível a alguma cota (ação afirmativa) e a escolha do curso desejado.
Explore cenários diversos e simule diferentes combinações de dados para obter as possibilidades de aprovação, por meio de uma interface intuitiva.
